## 19. 你该如何记录一个仓库

每看一个仓库，复制下面模板填一遍。

```text
仓库名：
仓库地址：

一句话说明：

项目类型：
[ ] 前端
[ ] 后端
[ ] Python/AI
[ ] MCP
[ ] 文档
[ ] 数据库
[ ] Docker/部署
[ ] CLI 工具
[ ] SDK
[ ] 其他：

主要语言：

关键文件：
- README：
- package.json / pyproject.toml：
- Dockerfile / docker-compose：
- .env.example：
- docs：

怎么安装：

怎么启动：

怎么测试：

我现在最该看懂的 3 个文件：
1.
2.
3.

暂时不用看的内容：

卡住的问题：

需要请 Codex 帮忙的问题：
```
